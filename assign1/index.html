Overloading the Internet

This website project would focus on the fact that the internet is overflowing with information, recently more now than ever. We are seeing more and more web pages, Instagram stories and archives being slowly deleted. For this project I’d want to focus on the impermanence of the internet but also how easily something can be truly removed from existence despite what we’ve been told about the internet being forever.

Vanishing Point: A Browser That Slowly Deletes Itself

This concept focuses on a website that erases its content over time. Every time a person lands on the website, it speeds up the process of decay. Visitors can try to save pieces of information but can never save the whole website. I feel that this concept is not only symbolic of the impermanence of the web but also creates a sort of critic of democracy as someone can choose to not save something you did to save your information. To further this concept the content of this webpage could be self-uploaded, like a forum, this would make the users face the uncomfortable reality that you cannot save everything.
